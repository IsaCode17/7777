Описание

CSS Свойство font-size определяет размер шрифта. Это свойство также используется для вычисления размера em, ex и других относительных единиц. Подробнее: {{cssxref("<length>")}}.

{{EmbedInteractiveExample("pages/css/font-size.html")}}

Синтаксис

/* значения в <абсолютных размерах> */
font-size: xx-small;
font-size: x-small;
font-size: small;
font-size: medium;
font-size: large;
font-size: x-large;
font-size: xx-large;

/* значения в <относительных размерах> */
font-size: larger;
font-size: smaller;

/* <значения длины> */
font-size: 12px;
font-size: 0.8em;

/* <процентные значения> */
font-size: 80%;

/* Глобальные значения */
font-size: inherit;
font-size: initial;
font-size: unset;

Свойство font-size устанавливается одним из следующих способов:

  • Ключевым словом из перечня абсолютных значений или относительных значений
  • Как <значение длины> или <процентное значение>, по отношению к размеру родительского элемента.

Значения

xx-small, x-small, small, medium, large, x-large, xx-large
Набор ключевых слов абсолютных значений, по отношению к пользовательскому размеру шрифта по умолчанию (им считается medium - среднее).
larger, smaller
Больше (larger) или меньше (smaller). Ключевые слова для относительного размера. Шрифт будет больше или меньше по отношению в размеру шрифта родительского элемента.Примерно на такое же соотношение, которое используется в ключевых словах абсолютного размера выше.
{{cssxref("<length>")}}

Положительное значение длины {{cssxref("<length>")}}. Для большинства единиц измерения, зависимых от шрифта (таких как em и ex), размер шрифта будет зависеть от размера шрифта родительских элементов.

Для единиц измерения, зависимых от шрифта, которые зависят от корневых единиц (таких как rem), размер шрифта будет коррелироваться по отношению к шрифту, используемому корневым элементом {{HTMLElement("html")}} (root).

<процентные значения>
Положительное процентное {{cssxref("<percentage>")}} значение по отношению к размеру шрифта родительского элемента.

Примечание: Для обеспечения максимальной совместимости обычно лучше использовать значения, относящиеся к размеру шрифта пользователя по умолчанию.

Возможные подходы

Существуют разные способы задания размера шрифта. С помощью ключевых слов или с помощью числовых значений для размера пикселей или размера ems. Выберите подходящий метод в зависимости от потребностей конкретной веб-страницы.

Ключевые слова

Ключевые слова — это хороший способ задать размер шрифта. Установив размер шрифта ключевого слова в элементе body, вы можете установить относительный размер шрифта везде на странице, что даст вам возможность легко увеличивать или уменьшать размер шрифта на всей странице соответственно.

Пиксели

Установка размера шрифта в пикселях (px) хороший выбор, когда вам нужна точность пикселей. Значение px является статическим. Это независимый от операционной системы и кросс-браузерный способ буквально указать браузерам отображать текст точно с указанным количеством пикселей. Результаты могут немного различаться в разных браузерах, поскольку они могут использовать разные алгоритмы для достижения одинакового эффекта.

Параметры font-size также можно использовать в комбинациях. Например, если для родительского элемента установлено значение 16px, а для дочернего larger, то размер текста дочернего элемента будет больше, чем родительского.

Примечание: Определение размера шрифта в пикселях не лучшим образом сказывается на доступности, потому что пользователь не может изменить размер шрифта из браузера. (Например, пользователи с ограниченным зрением могут захотеть установить размер шрифта намного больше, чем размер, выбранный веб-дизайнером.) Поэтому избегайте использования пикселей для размеров шрифта, если вы хотите создать дизайн доступный пользователям с ограниченными возможностями.

Em

Еще один способ установить размер шрифта - это использование em. Значение em является динамическим. При определении свойства font-size, em равно размеру шрифта, который применяется к родителю рассматриваемого элемента. Если вы нигде на странице не установили font-size, то это будет значение браузера по умолчанию, которое, вероятно, составляет 16px. Итак, по умолчанию 1em = 16px, а 2em = 32px. Если вы установите 20px для элемента body, то 1em = 20px, а 2em = 40px. Обратите внимание, что значение 2, по сути, является множителем текущего размера em.

Чтобы вычислить эквивалент em для любого требуемого значения пикселя, вы можете использовать эту формулу:

em = желаемое значение элемента в пикселях / размер шрифта родительского элемента в пикселях

Например, предположим, что размер шрифта основной части страницы установлен 1em, а стандарт браузера 1em = 16px; если размер шрифта, который вы хотите, составляет 12px, вы должны указать 0.75em (поскольку 12/16 = 0.75). Точно так же, если вы хотите чтобы размер шрифта был 10px, укажите 0.625em (10/16 = 0.625); для 22px укажите 1.375em (22/16).

Популярный метод — установить размер шрифта body в 62.5% (то есть 62.5% от значения по умолчанию 16px), что соответствует 10px или 0.625em. В таком случае вы можете устанавливать размер шрифта для любых элементов, используя единицы em, с легко запоминающимся преобразованием, разделив значение px на 10. Таким образом, 6px = 0.6em, 8px = 0.8em, 12px = 1.2em, 14px = 1.4em, 16px = 1.6em. Например:

body {
  font-size: 62.5%; /* font-size 1em = 10px в настройках браузера по умолчанию */
}
span {
  font-size: 1.6em; /* 1.6em = 16px */
}

em — очень полезная единица в CSS, поскольку она автоматически адаптирует свою длину в зависимости от шрифта, который выберет пользователь.

Следует иметь в виду один важный факт: значения em составные. Возьмите следующий HTML-код и примените его к предыдущему CSS:

<div>
<span>Внешний <span>внутренний</span> внешний</span>
</div>

Результат:

Предполагая, что font-size браузера по умолчанию составляет 16px, слова «внешний» будут отображаться с размером 16px, а слово «внутренний» — 25.6px. Это связано с тем, что font-size внутреннего элемента span составляет 1.6em относительно font-size его родителя, который, в свою очередь, зависит от font-size своего родителя. Это часто называют смешиванием.

Rem

rem были разработаны для того, чтобы обойти проблему смешивания. Значения rem зависят от размера шрифта корневого элемента html, а не родительского элемента. Другими словами, rem позволяет вам указать размер шрифта относительным образом, не влияя на размер родителя, тем самым избегая смешивания.

Приведенный ниже CSS почти идентичен предыдущему примеру. Единственным исключением является то, что единица измерения была изменена на rem.

html {
  font-size: 62.5%; /* font-size 1em = 10px в настройках браузера по умолчанию */
}
span {
  font-size: 1.6rem;
}

Затем мы применяем этот CSS к тому же HTML, который выглядит так:

<span>Внешний <span>внутренний</span> внешний</span>

{{EmbedLiveSample("Rems", 400, 40)}}

В этом примере все слова «внешний внутренний внешний» отображаются с размером 16px (при условии, что размер шрифта браузера по умолчанию равен 16px).

Примеры

Пример 1

/* Устанавливает "очень большой" размер
 * текста для элемента p. */
p { font-size: xx-large }

/* Уставновливает размер текста h1 (заголовок 1 уровня) в 2.5 раза
 * больше чем текст вокруг него. */
h1 { font-size: 250% }

/* Устанавливает размер текста, заключенного в теге span, равным 16px */
span { font-size: 16px; }

Пример 2

.small {
	font-size: xx-small;
}
.larger {
	font-size: larger;
}
.point {
	font-size: 24pt;
}
.percent {
	font-size: 200%;
}

<h1 class="small">Small H1</h1>
<h1 class="larger">Larger H1</h1>
<h1 class="point">24 point H1</h1>
<h1 class="percent">200% H1</h1>

Примечание

Единицы измерения em и ex свойства {{Cssxref("font-size")}} зависят от размера шрифта родительского элемента (в отличие от всех других свойств, которые зависят от размера шрифта элемента). Это означает, что значения em и проценты делают то же самое для {{Cssxref("font-size")}}.
