{{CSSRef}}

Описание

Свойство CSS border-image-source определяет {{cssxref("<image>")}} который будет использоваться вместо стиля границы. Если для этого свойства установлено значение none, то вместо него используется стиль, определенный параметром {{cssxref("border-style")}}.

Предупреждение: Хотя с этим свойством CSS можно использовать любой {{cssxref("<image>")}} , поддержка браузеров по-прежнему ограничена, и некоторые браузеры поддерживают только изображения, определенные с использованием функциональной нотации url() .

{{cssinfo}}

Синтаксис

/* без изображения границы, используется указанный стиль границы */
border-image-source: none;

/* image.jpg используется в качестве изображения */
border-image-source: url(image.jpg);

/* В качестве изображения используется градиент */
border-image-source: linear-gradient(to top, red, yellow);

/* Глобальные значения */
border-image-source: inherit;
border-image-source: initial;
border-image-source: unset;

Значения

none
Указывает, что для границ не используется изображение. Вместо этого используется стиль, определенный {{cssxref("border-style")}}.
<image>
Ссылка на изображение, которое будет использоваться в отрисовке границы.
