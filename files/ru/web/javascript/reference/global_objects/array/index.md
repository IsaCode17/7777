Массив (**`Array`**) в JavaScript — это глобальный объект, который используется для создания массивов — высокоуровневых спископодобных объектов.

## Описание

Массивы — это спископодобные объекты, чьи прототипы содержат методы для операций обхода и изменения массива. Ни размер JavaScript-массива, ни типы его элементов не являются фиксированными. Поскольку длина массива может измениться в любое время, а данные могут храниться в несоприкасающихся частях массива, то в JavaScript нет гарантии, что массив окажется плотным. Это зависит от того, как разработчик решает пользоваться им. Обычно эти особенности массивов удобны, но если они не подходят под вашу ситуацию, рассмотрите вариант использования типизированных массивов.

Массивы не могут использовать строки в качестве идексов элементов (как в [ассоциативном массиве](https://ru.wikipedia.org/wiki/%D0%90%D1%81%D1%81%D0%BE%D1%86%D0%B8%D0%B0%D1%82%D0%B8%D0%B2%D0%BD%D1%8B%D0%B9_%D0%BC%D0%B0%D1%81%D1%81%D0%B8%D0%B2)), вместо этого они должны использовать целые числа. Попытка назначить или получить элемент с помощью чего-либо, кроме целых чисел, через [скобочную](/ru/docs/Web/JavaScript/Guide/Working_with_Objects#%D0%BE%D0%B1%D1%8A%D0%B5%D0%BA%D1%82%D1%8B_%D0%B8_%D1%81%D0%B2%D0%BE%D0%B9%D1%81%D1%82%D0%B2%D0%B0) (или [точечную](ru/docs/Web/JavaScript/Reference/Operators/Property_Accessors)) нотацию не приведет к изменению или получению элемента из самого массива. Вместо этого произойдет изменение или получение переменной, связанной с [набором свойств объекта](/ru/docs/Web/JavaScript/Data_structures#%D1%81%D0%B2%D0%BE%D0%B9%D1%81%D1%82%D0%B2%D0%B0) этого массива. Набор свойств объекта массива и список элементов массива существуют отдельно друг от друга, и [операции обхода и изменения](/ru/docs/Web/JavaScript/Guide/Indexed_collections#%D0%BC%D0%B5%D1%82%D0%BE%D0%B4%D1%8B_array) массива не могут быть применены к этим именованным свойствам.

### Распространённые операции

**Создание массива**

```js
let fruits = ['Яблоко', 'Банан']

console.log(fruits.length)
// 2
```

**Получение элемента массива с помощью индекса**

```js
let first = fruits[0]
// Яблоко

let last = fruits[fruits.length - 1]
// Банан
```

**Перебор элементов массива**

```js
fruits.forEach(function(item, index, array) {
  console.log(item, index)
})
// Яблоко 0
// Банан 1
```

**Добавление элемента в конец массива**

```js
let newLength = fruits.push('Апельсин')
// ["Яблоко", "Банан", "Апельсин"]
```

**Удаление элемента из конца массива**

```js
let last = fruits.pop() // удалит апельсин (из конца)
// ["Яблоко", "Банан"]
```

**Удалание элемента из начала массива**

```js
let first = fruits.shift() // удалит яблоко (из начала)
// ["Банан"]
```

**Добавление элемент в начало массива**

```js
let newLength = fruits.unshift('Клубника') // добавит в начало
// ["Клубника", "Банан"]
```

**Поиск индекса элемента в массиве**

```js
fruits.push('Манго')
// ["Клубника", "Банан", "Манго"]

let pos = fruits.indexOf('Банан')
// 1
```

**Удаление элемента с помощью его индекса**

```js
let removedItem = fruits.splice(pos, 1) // так можно удалить элемент

// ["Клубника", "Манго"]
```

**Удаление элементов, начиная с определённого индекса**

```js
let vegetables = ['Капуста', 'Репа', 'Редиска', 'Морковь']
console.log(vegetables)
// ["Капуста", "Репа", "Редиска", "Морковь"]

let pos = 1
let n = 2

let removedItems = vegetables.splice(pos, n)
// так можно удалить элементы: n определяет количество элементов, которые нужно удалить,
// начиная с индекса, определённого в pos, и до конца массива

console.log(vegetables)
// ["Капуста", "Морковь"] (изначальный массив изменён)

console.log(removedItems)
// ["Репа", "Редис"]
```

**Копирование массива**

```js
let shallowCopy = fruits.slice() // вот так можно скопировать массив
// ["Клубника", "Манго"]
```

### Доступ к элементам массива

Массивы в JavaScript индексируются с нуля. Индекс первого элемента — `0`, а индекс последнего равен значению свойства массива {{jsxref("Array.length", "length")}} минус `1`.

Использование неверного номера индекса возвращает `undefined`.

```js
let arr = ['первый элемент', 'второй элемент', 'последний элемент']
console.log(arr[0])              // напечатает 'первый элемент'
console.log(arr[1])              // напечатает 'второй элемент'
console.log(arr[arr.length - 1]) // напечатает 'последний элемент'
```

Элементы массива являются свойствами, точно такими же, как, например, свойство `toString` (для точности: `toString()` — это метод). Однако попытка получить элемент массива по подобному имени его свойства приведёт к синтаксической ошибке, поскольку оно не является допустимым в JavaScript:

```js
console.log(arr.0) // синтаксическая ошибка
```

Это не особенность JavaScript-массивов или их свойств. В JavaScript к свойствам, начинающимся с цифры, невозможно обратиться посредством точечной нотации, только с помощью скобочной нотации. 

Например, если у вас есть объект со свойством `3d`, вы сможете обратиться к нему только посредством скобочной нотации.

```js
let years = [1950, 1960, 1970, 1980, 1990, 2000, 2010]
console.log(years.0)   // синтаксическая ошибка
console.log(years[0])  // работает как положено
```

```js
renderer.3d.setTexture(model, 'character.png')     // синтаксическая ошибка
renderer['3d'].setTexture(model, 'character.png')  // работает как положено
```

В последнем примере `'3d'` *необходимо* было заключить в кавычки (потому что первый символ — это цифра). Индексы тоже можно заключать в кавычки (например, `years['2']` вместо `years[2]`), однако в этом нет необходимости.

`2` в `years[2]` неявно приведено к строке движком JavaScript через метод преобразования `toString`. В результате `'2'` и `'02'` будут ссылаться на два разных элемента в объекте `years`, и следующий пример выведет `true`:

```js
console.log(years['2'] != years['02'])
```

### Взаимосвязь свойства length с числовыми свойствами

Свойство массивов {{jsxref("Array.length", "length")}} взаимосвязано с числовыми свойствами.

Некоторые встроенные методы массива (например, {{jsxref("Array.join", "join")}}, {{jsxref("Array.slice", "slice")}}, {{jsxref("Array.indexOf", "indexOf")}} и т.д.) учитывают значение свойства {{jsxref("Array.length", "length")}} при своём вызове.

Другие методы (например, {{jsxref("Array.push", "push")}}, {{jsxref("Array.splice", "splice")}} и т.д.) в результате своей работы также обновляют свойство {{jsxref("Array.length", "length")}} массива.

```js
const fruits = []
fruits.push('банан', 'яблоко', 'персик')

console.log(fruits.length) // 3
```

При установке свойства в массиве, если свойство имеет действительный индекс и этот индекс выходит за пределы текущих границ массива, движок соответствующим образом обновит свойство {{jsxref("Array.length", "length")}}:

```js
fruits[5] = 'манго'
console.log(fruits[5])            // 'манго'
console.log(Object.keys(fruits))  // ['0', '1', '2', '5']
console.log(fruits.length)        // 6
```

Увеличение свойства {{jsxref("Array.length", "length")}}.

```js
fruits.length = 10
console.log(fruits)              // ['банан', 'яблоко', 'персик', empty x 2, 'манго', empty x 4]
console.log(Object.keys(fruits)) // ['0', '1', '2', '5']
console.log(fruits.length)       // 10
console.log(fruits[8])           // undefined
```

Уменьшение свойства {{jsxref("Array.length", "length")}}, однако, удаляет элементы.

```js
fruits.length = 2
console.log(Object.keys(fruits)) // ['0', '1']
console.log(fruits.length)       // 2
```

Более подробно эта тема освещена на странице, посвящённой свойству {{jsxref("Array.length")}}.

### Создание массива с использованием результата сопоставления

Результатом сопоставления регулярного выражения ({{jsxref("RegExp")}}) строке может стать JavaScript-массив. У такого массива есть свойства и элементы, предоставляющие информацию о сопоставлении. Подобные массивы возвращаются методами {{jsxref("RegExp.exec")}} и {{jsxref("String.match")}}.

Чтобы было проще понять, откуда и какие появились свойства и элементы, посмотрите следующий пример и обратитесь к таблице ниже

```js
// Сопоставляется с одним символом d, за которым следует один
// или более символов b, за которыми следует один символ d
// Запоминаются сопоставившиеся символы b и следующий за ними символ d
// Регистр игнорируется

const myRe = /d(b+)(d)/i
const myArray = myRe.exec('cdbBdbsbz')
```

## Методы экземпляров

- {{jsxref("Array.prototype.at()")}}{{Experimental_Inline}}
  - : Возвращает элемент массива по заданному индексу. Также принимает отрицательные целые числа: в таком случае отсчет идет от последнего элемента массива.
- {{jsxref("Array.prototype.concat()")}}
  - : Возвращает новый массив, который является результатом соединения массива с другими массивами и/или значениями.
- {{jsxref("Array.prototype.copyWithin()")}}
  - : Копирует последовательность элементов массива внутрь массива.
- {{jsxref("Array.prototype.entries()")}}
  - : Возвращает новый объект _итератора массива_, который содержит пары ключ / значение для каждого индекса в массиве.
- {{jsxref("Array.prototype.every()")}}
  - : Возвращает `true`, если каждый элемент массива удовлетворяет условиям, заданным в тестирующей функции.
- {{jsxref("Array.prototype.fill()")}}
  - : Заполяет все элементы массива от первого до последнего индекса одним значением.
- {{jsxref("Array.prototype.filter()")}}
  - : Возвращает новый массив, содержащий все прошедшие проверку элементы массива, на котором вызвана передаваемая функция.
- {{jsxref("Array.prototype.find()")}}
  - : Возвращает первый найденный элемент массива, который удовлетворяет условию в передаваемой функции, или `undefined`, если таких нет.
- {{jsxref("Array.prototype.findIndex()")}}
  - : Возвращает индекс элемента массива, удовлетворяющего условию в передаваемой функции, или `-1`, если таких нет.
- {{jsxref("Array.prototype.flat()")}}
  - : Возвращает новый массив с рекурсивно присоединёнными элементами вложенных подмассивов влоть до заданного уровня вложенности.
- {{jsxref("Array.prototype.flatMap()")}}
  - : Возвращает новый массив, сформированный путём применения заданной функции к каждому элементу начального массива и последующего уменьшения уровня вложенности на один.
- {{jsxref("Array.prototype.forEach()")}}
  - : Вызывает заданную функцию для каждого элемента массива.
- {{jsxref("Array.prototype.includes()")}}
  - : Определяет, если массив содержит заданное значение, и возвращает `true` или `false` соответственно.
- {{jsxref("Array.prototype.indexOf()")}}
  - : Возвращает первый (наименьший) индекс элемента, по которому он может быть найден в массиве, или `-1`, если такого нет.
- {{jsxref("Array.prototype.join()")}}
  - : Соединяет все элементы массива в строку.
- {{jsxref("Array.prototype.keys()")}}
  - : Возвращает новый объект _итератора массива_, который содержит ключи каждого индекса массива.
- {{jsxref("Array.prototype.lastIndexOf()")}}
  - : Возвращает последний (наибольший) индекс элемента, по которому он может быть найден в массиве, или `-1`, если такого нет.
- {{jsxref("Array.prototype.map()")}}
  - : Возвращает новый массив, содержащий результат выполнения функции для каждого элемента исходного массива.
- {{jsxref("Array.prototype.pop()")}}
  - : Удаляет последний элемент массива и возвращает его значение.
- {{jsxref("Array.prototype.push()")}}
  - : Добавляет один или более элементов в конец массива и возвращает новую длину (`length`) массива.
- {{jsxref("Array.prototype.reduce()")}}
  - : Применяет функцию к аккумулятору и каждому элементу массива (слева направо), чтобы привести его к одному результирующему значению.
- {{jsxref("Array.prototype.reduceRight()")}}
  - : Применяет функцию к аккумулятору и каждому элементу массива (справа налево), чтобы привести его к одному результирующему значению.
- {{jsxref("Array.prototype.reverse()")}}
  - : Обращает порядок следования элементов массива _на месте_. (Первый элемент становится последний, последний — первым.) 
- {{jsxref("Array.prototype.shift()")}}
  - : Удаляет первый элемент массива и возвращает его значение.
- {{jsxref("Array.prototype.slice()")}}
  - : Извлекает часть исходного массива и возвращает ее в виде нового массива.
- {{jsxref("Array.prototype.some()")}}
  - : Возвращает `true`, если хотя бы один элемент массива удовлетворяет условию в заданной функции.
- {{jsxref("Array.prototype.sort()")}}
  - : Сортирует элементы на месте и возвращает массив.
- {{jsxref("Array.prototype.splice()")}}
  - : Добавляет и/или удаляет элементы из массива.
- {{jsxref("Array.prototype.toLocaleString()")}}
  - : Возвращает локале-зависимое строковое представление элементов массива. Перезаписывает метод {{jsxref("Object.prototype.toLocaleString()")}}.
- {{jsxref("Array.prototype.toString()")}}
  - : Возвращает строковое представление элементов массива. Перезаписывает метод {{jsxref("Object.prototype.toLocaleString()")}}.
- {{jsxref("Array.prototype.unshift()")}}
  - : Добавляет один или более элементов в начало массива и возвращает новую длину (`length`) массива.
- {{jsxref("Array.prototype.values()")}}
  - : Возвращает новый объект _итератора массива_, который содержит значения для каждого индекса массива.
- {{jsxref("Array.prototype.@@iterator()", "Array.prototype[@@iterator]()")}}
  - : Возвращает новый объект _итератора массива_, который содержит значения для каждого индекса массива.

## Примеры

### Создание массива

Следующий пример создает массив `msgArray` с длиной `0`, а затем задает значения для `msgArray[0]` и `msgArray[99]`, тем самым меняя длину (`length`) массива на `100`.

```js
let msgArray = []
msgArray[0] = 'Привет'
msgArray[99] = 'мир'

if (msgArray.length === 100) {
  console.log('Длина массива 100.')
}
```

### Создание двумерного массива

Следующий код создаёт шахматную доску в виде двумерного массива строк. Затем он перемещает пешку путём копирования символа `'p'` в позиции `board[6][4]` на позицию `board[4][4]`. Старая позиция `[6][4]` затирается пустым местом.

```js
let board = [
  ['R','N','B','Q','K','B','N','R'],
  ['P','P','P','P','P','P','P','P'],
  [' ',' ',' ',' ',' ',' ',' ',' '],
  [' ',' ',' ',' ',' ',' ',' ',' '],
  [' ',' ',' ',' ',' ',' ',' ',' '],
  [' ',' ',' ',' ',' ',' ',' ',' '],
  ['p','p','p','p','p','p','p','p'],
  ['r','n','b','q','k','b','n','r'] ]

console.log(board.join('\n') + '\n\n')

// Переместить пешку на две клетки вперёд
board[4][4] = board[6][4]
board[6][4] = ' '
console.log(board.join('\n'))
```

Ниже показан вывод:

```plain
R,N,B,Q,K,B,N,R
```
