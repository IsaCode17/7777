<span class="seoSummary">CSS препроцессор (<strong>CSS preprocessor</strong>) - </span>это программа, которая имеет свой собственный синтаксис (<span class="seoSummary">{{Glossary("syntax")}})</span>, но может сгенерировать из него<span class="seoSummary"> {{Glossary("CSS")}} код .</span> Существует множество препроцессоров. Большинство из них расширяет возможности чистого CSS, добавляя такие опции как: примеси, вложенные правила, селекторы наследования и др. Эти особенности облегчают работу с CSS: упрощают чтение кода и его дальнейшую поддержку.

Для использования CSS препроцессора нужно установить CSS компилятор на ваш веб-сервер ({{Glossary("server")}}).
