Запрещённое имя заголовка ответа это имя [HTTP заголовка](/en-US/docs/Web/HTTP/Headers) (\`<code title="">Set-Cookie</code>\` или \`<code title="">Set-Cookie2</code>\`), который не может быть изменён программно.
