{{APIRef}}

Resumen

La propiedad opener, de la interfaz {{domxref("Window")}}, devuelve una referencia de la ventana que abrió otra ventana, ya fuese con {{domxref("Window.open", "open()")}}, o navegando un enlace que tuviese el atributo {{htmlattrxref("target", "a")}} con el valor "_blank". La referencia es un objeto {{domxref("Window")}}.

En otras palabras, si la ventana A abre la ventana B, y se escribe en la consola de la ventana B window.opener, se devuelve un objeto {{domxref("Window")}} de la ventana A.

Sintaxis

const ventanaReferida = window.opener;

Ejemplo

  //Código para la ventana que fue abierta
  if (window.opener != null) {
   const ventanaReferida = window.opener;
   ventanaReferida.document.write("Este es el nuevo contenido de la ventana referida");
   ventanaReferida.alert("Se ha sobrescrito el contenido");
  }

Notas

Por motivos de seguridad, es posible que esta propiedad sólo devuelva la ventana referida si ambas páginas tienen el mismo origen ({{HTTPHeader("Origin")}}); devolviendo {{jsxref("null")}} si se intenta usar en una página externa.

El navegador de Windows Phone no soporta window.opener. Tampoco es soportado en Internet Explorer si la ventana nueva se encuentra en una zona de seguridad distinta.
