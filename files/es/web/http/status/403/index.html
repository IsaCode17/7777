El error 403 o “403 Forbidden” es un código de respuesta HTTP que indica que el servidor ha recibido y ha entendido la petición, pero que no puede autorizar la respuesta.
 
La diferencia entre el error {{HTTPStatus("401")}} y el error 403, es que el 401 normalmente se soluciona autentificándonos, mientras que el 403 suele ser causado por un problema real en la aplicación web o en el entorno (servidor o configuración del servidor).
 
Un error 403 en una web no puede solucionarlo el usuario, sino que es el webmaster o administrador del servidor quien tiene que solucionarlo.
