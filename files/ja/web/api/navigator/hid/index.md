読み取り専用の **`Navigator.hid`** プロパティは、{{domxref("HID")}} オブジェクトを返します。このオブジェクトは、HID デバイスへの接続およびアタッチした HID デバイスのリストアップを行うメソッドと、HID デバイスの接続を扱うイベントハンドラーを提供します。

## 値

{{domxref("HID")}} オブジェクトです。
