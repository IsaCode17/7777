A propriedade fileName contém o caminho para o arquivo que levantou este erro.

Descrição

Esta propriedade não-padrão contém o caminho para o ficheiro que levantou este erro. Se chamado a partir de um contexto de depuração, as Ferramentas de Desenvolvimento Firefox, por exemplo, "debugger eval code" é devolvido.

Exemplos

Usando fileName

var e = new Error('Could not parse input');
lançar e;
// e.fileName could look like "file:///C:/example.html"

Especificações

Não faz parte de nenhuma norma.
