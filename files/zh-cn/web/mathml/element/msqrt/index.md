[MathML](/zh-CN/docs/Web/MathML) **`<msqrt>`** 元素用于表示平方根（不显示根指数）。它只接受一个参数，语法如下：`<msqrt> base </msqrt>`。

## 属性

该元素接受[全局 MathML 属性](/zh-CN/docs/Web/MathML/Global_attributes)。

## 示例

```html
<math display="block">
  <msqrt>
    <mi>x</mi>
  </msqrt>
</math>
```
