{{ ApiRef() }}

打开打印对话框，以打印当前文档。如果调用此函数时文档仍在加载中，那么文档将在打开打印对话框之前完成加载。当打印对话框打开时，此方法将阻塞。

## 语法

```js-nolint
print()
```

### 参数

无

### 返回值

无（[`undefined`](/zh-CN/docs/Web/JavaScript/Reference/Global_Objects/undefined)）
