**`HTMLElement.accessKey`** 属性设置了用户按下的按键，以跳转到给定的元素。

> **备注：** 由于 `HTMLElement.accessKey` 与浏览器中已存在的按键绑定存在多重冲突，属性很少被使用。为了解决这个问题，浏览器会在按下其他“限定”键（例如 <kbd>Alt</kbd> + accesskey）时实现访问键行为。
