MediaQueryListEvent() 构造函数创建出一个新的 {{DOMxRef("MediaQueryListEvent")}} 对象

语法

new MediaQueryListEvent(type)
new MediaQueryListEvent(type, options)

参数

type
一个带有该事件名字的字符串。该字符串是大小写敏感的，且浏览器通常将其设置为 change。
options {{optional_inline}}
一个对象，在 {{domxref("Event/Event", "Event()")}}已经定义好的属性基础上，其可以有如下的属性：
media {{optional_inline}}
一个字符串，代表着一个序列化的媒体查询。默认值为 ""
matches {{optional_inline}}
一个布尔值，代表着匹配媒体查询状态。如果匹配该值为true，反之为false。默认值为 false

返回值

一个新的{{DOMxRef("MediaQueryListEvent")}} 对象

示例

var media = '(max-width: 600px)';
var matches = true;

var myMediaQueryListEvent = new MediaQueryListEvent("change", {media, matches});
