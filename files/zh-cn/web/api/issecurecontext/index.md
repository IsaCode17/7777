{{APIRef()}}

全局 **`isSecureContext`** 只读属性返回一个布尔值，指示当前[上下文是否是安全的](/zh-CN/docs/Web/Security/Secure_Contexts)，安全（`true`）或不安全（`false`）。

## 值

一个布尔值，`true` 或 `false`。
