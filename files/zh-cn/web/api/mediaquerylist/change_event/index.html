{{APIRef("CSSOM")}}

当媒体查询的支持情况改变时，{{DOMxRef("MediaQueryList")}} 接口的change事件触发。

语法

在像 {{domxref("EventTarget.addEventListener", "addEventListener()")}} 这样的方法中使用该事件名字，或者设置一个事件处理函数属性 

addEventListener('change', event => { });

onchange = event => { };

事件类型

{{domxref("MediaQueryListEvent")}} 类型。 继承于 {{domxref("Event")}}。

{{InheritanceDiagram("MediaQueryListEvent")}}

事件属性

MediaQueryListEvent 接口的属性继承自它的父接口，{{domxref("Event")}}。

{{DOMxRef("MediaQueryListEvent.matches")}}{{ReadOnlyInline}}
一个布尔值，如果 {{DOMxRef("document")}} 目前匹配媒体查询列表，该值为true，反之为false。
{{DOMxRef("MediaQueryListEvent.media")}}{{ReadOnlyInline}}
一个字符串，代表着一个序列化的媒体查询。

示例

var mql = window.matchMedia('(max-width: 600px)');

function screenTest(e) {
  if (e.matches) {
    /* 视口小于或等于600像素 */
    console.log('This is a narrow screen — less than 600px wide.')
  } else {
    /* 视口大于600像素 */
    console.log('This is a wide screen — more than 600px wide.')
  }
}

mql.addListener(screenTest);
