{{APIRef("CSSOM")}}

{{DOMxRef("MediaQueryList")}} 接口的matches只读属性是一个布尔值，如果 {{DOMxRef("document")}} 当前匹配媒体查询列表，该属性返回true，否则返回false。

通过侦听发生在 {{DOMxRef("MediaQueryList")}} 的 {{domxref("MediaQueryList.change_event", "change")}} 事件，当matches的值发生变化时，你将会被告知。

值

一个布尔值，如果 {{DOMxRef("document")}} 目前匹配媒体查询列表，该值为true，反之为false。

示例

这个示例通过使用orientation媒体查询特性来创建一个媒体查询，检测视口横纵方向的变化 

  function addMQListener(mq, callback) {
    if (mq.addEventListener) {
      mq.addEventListener("change", callback);
    } else {
      mq.addListener(callback);
    }
  }
  
  addMQListener(window.matchMedia("(orientation:landscape)"),
    event => {
      if (event.matches) {
        /* 现在处于横向 */
      } else {
        /* 现在处于纵向 */
      }
    }
  );
