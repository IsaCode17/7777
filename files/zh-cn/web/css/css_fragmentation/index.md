**CSS 片段**模块定义了当内容在多个[页面](/zh-CN/docs/Web/CSS/CSS_paged_media)、区域或[列](/zh-CN/docs/Web/CSS/CSS_multicol_layout)中被分割（fragmented）时的显示方式。

当行内盒子被分割到多行上时，就会出现分片。当一个块在列布局容器内跨越多列，或在打印时跨越分页符时，也会出现这种情况。元素的每个呈现片段都称为*分片*。
