当左边的操作数小于右边的操作数时，小于运算符（<）返回 true，否则返回 false。

{{EmbedInteractiveExample("pages/js/expressions-less-than.html")}}

语法

 x < y

描述

操作数之间按照 Abstract Relational Comparison 算法进行比较，大致可概括如下：

  • 首先，用带有 hint 参数的 Symbol.ToPrimitive将对象转换成原始值。
  • 如果两个值都是字符串，则根据它们包含的 Unicode 代码位的值，将它们作为字符串进行比较。
  • 否则 Javascript 会尝试将非数字类型转换成数字值：
    • 布尔值 true 和 false 会被各自转换成 1 和 0。
    • null 会被转换成 0。
    • undefined 会被转换成 NaN。
    • 字符串根据它们包含的值进行转换，如果它们不包含数字值则会被转换成 NaN。
  • 如果任一值为 NaN，运算符会返回 false.
  • 其它情况下，值被作为数字值比较。

例子

字符串间的比较

console.log("a" < "b");        // true
console.log("a" < "a");        // false
console.log("a" < "3");        // false

字符串和数字的比较

console.log("5" < 3);          // false
console.log("3" < 3);          // false
console.log("3" < 5);          // true

console.log("hello" < 5);      // false
console.log(5 < "hello");      // false

console.log("5" < 3n);         // false
console.log("3" < 5n);         // true

数字间的比较

console.log(5 < 3);            // false
console.log(3 < 3);            // false
console.log(3 < 5);            // true

数字与 BigInt 数据的比较

console.log(5n < 3);           // false
console.log(3 < 5n);           // true

Boolean, null, undefined, NaN 的比较

console.log(true < false);     // false
console.log(false < true);     // true

console.log(0 < true);         // true
console.log(true < 1);         // false

console.log(null < 0);         // false
console.log(null < 1);         // true

console.log(undefined < 3);         // false
console.log(3 < undefined);         // false

console.log(3 < NaN);          // false
console.log(NaN < 3);          // false
