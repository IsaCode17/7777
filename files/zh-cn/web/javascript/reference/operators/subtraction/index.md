减法运算符（`-`）对两个操作数进行减法，产生它们的差值。

{{EmbedInteractiveExample("pages/js/expressions-subtraction.html")}}

## 语法

```js-nolint
x - y
```

## 示例

### 减法与数值

```js
5 - 3     // 2
3 - 5     // -2
```

### 减法与非数值

```js
'foo' - 3 // NaN
```
