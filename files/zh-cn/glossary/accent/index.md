---
title: 点缀色
slug: Glossary/Accent
---

**点缀色**（accent）通常为一种明亮的颜色，与颜色方案中更实用的背景和前景色形成对比。这些颜色存在于很多平台（但不是所有平台）的视觉样式中。

在 web 上，点缀色有时用于 {{HTMLElement("input")}} 元素中控件的活跃部分，例如勾选的[复选框](/zh-CN/docs/Web/HTML/Element/input/checkbox)的背景。

## 参见

### 与点缀色相关的 CSS

为设置指定元素的点缀色，可以将元素的 CSS {{CSSXref("accent-color")}} 属性设置为合适的 {{CSSXref("&lt;color&gt;")}} 值。
