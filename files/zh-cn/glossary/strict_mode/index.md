JavaScript 的**严格模式**是使用受限制的 JavaScript 的一种方式，从而隐式地退出“{{Glossary("Sloppy_mode", "正常模式")}}”。严格模式不仅仅是一个子集：这种模式有意地与普通情形下的代码有所区别。

通过在脚本文件/函数开头添加 `"use strict";` 声明，即可启用严格模式。
