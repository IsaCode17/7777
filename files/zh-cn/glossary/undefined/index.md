{{GlossarySidebar}}

**`undefined`** 是一个 {{Glossary("primitive", "原始值")}}，自动分配给刚刚声明的 {{glossary("variable", "变量")}}，或者没有实际 {{Glossary("Argument", "Arguments")}} 的形式参数。

## Example

```js
let x; //创建一个变量但是没有赋值

console.log(`x's value is ${x}`); //输出“x 的值是 undefined”
```
