**Web 性能**是从用户请求内容到用户的浏览器中显示请求的内容的时间，客观的渲染时间以及加载时间和运行时间的主观用户体验。

客观上，它是测量网页或 Web 应用程序下载、在用户的 Web 浏览器中绘制并变得响应和交互所需的时间，以毫秒为单位。它是每秒帧数和主线程不可用于用户交互的时间。主观上，它是用户感知到的用户请求内容和用户感觉到请求的内容可用和可用之间的时间之间的时间是否*感觉*慢或快。
