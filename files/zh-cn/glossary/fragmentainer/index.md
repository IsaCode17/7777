碎片容器（fragmentainer）在 [CSS 内容碎片化规范](https://www.w3.org/TR/css-break-3/)中定义如下：

> 一个包含部分（或全部）内容碎片化流的盒子，如页面盒子、列盒子或区域盒子。碎片化容器可以是预定义的，也可以根据需要生成。当可断内容在块方向上溢出碎片化容器时，它会在其碎片化上下文中分割到下一个容器中。

碎片化上下文存在于 CSS 分页媒体中，其中碎片容器是定义页面的盒子。在多列布局中，碎片容器是定义在多列容器上的列盒子。在 CSS 区域中，每个区域都是一个碎片容器。
