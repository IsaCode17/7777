欢迎来到 MDN Web Docs！在使用过程中，如果你遇到了问题或者对我们有些建议，这篇文章就很适合你。如果你有兴趣反馈你的意见，就可以逐步融入到 Mozilla Community 中来。提前向你表达我们的谢意！

你分享的任何发现与想法，都可以帮助改进这个网站，下面就来介绍几种分享的途径。

更新文档

在浏览 MDN 时，假如发现某篇文章有些问题，你完全可以试着自己动手去改正它。如果你对此感兴趣，这里有一篇参与指南供你参考。

MDN 的文件内容现在都存放在 GitHub 上的仓库中（翻译后的内容则存放在 Translated Content中)，由一群志愿者和拿薪水的员工共同管理。所以请放心大胆地参与到编辑中，会有人及时审核这些内容并帮助你修改它。

在 MDN 站内，也有许多文章可以为你提供帮助，比如：

    「初识 MDN」中，简要介绍了参与方法。
  • 在「为 MDN 做贡献」中，列出了许多等待参与的任务。

参与讨论

欢迎和我们交流！下面列出了一些途径，方便你和 MDN 中其他的参与者们更好地交流。

在线实时聊天

我们使用 Matrix 来进行实时交流，主要是关于 MDN 以及其中的内容。我们随时都欢迎你的加入！

MDN Web Docs

一些日常的讨论，像网站的使用问题、网站上的文字内容、参与网站志愿编辑工作等，都可以在这个聊天室进行。如果你对网站上的文章内容有疑问或建议，或者希望在网站上看到或亲手创作一篇某个主题的新文章，又或者只是想和网站的内容创作者们有更多的交流，这里是个非常好的去处。

在线论坛

MDN 的论坛上，大多是一些需要反复讨论的内容。如果你想发起这样的讨论，向我们的邮箱——mdn@mozilla-community.org——发送邮件即可。在论坛注册帐号后，你还可以选择通过邮件来提醒你帖子下的新内容。

问题反馈

文档问题

如果你在文档中发现了问题，却不知道怎么修改，可以向我们提交 issue。你遇到的任何问题，都可以在这里反馈。如果是翻译上的问题，向 Translated-content Repo 提交一个 issue 会更合适。下面是一些比较常见 issue 的主题：

  • 内容的简单修正
  • 请求修正某些错误的内容
  • 反馈文章中有不太合适的内容（比如垃圾信息或不合适的翻译）

如果你有时间，想试着自己来完善这个文档，我们非常欢迎。当然，能够反馈你发现的问题，对我们也很有帮助！

网站问题

假如发现了网站本身的问题，或者对网站的功能有任何的建议，你都可以直接联系 MDN 的开发团队。
