欢迎来到 MDN Web Docs！在浏览 MDN Web Docs 时，如果你遇到了问题，或者想对我们提些建议，哪怕只是对这个网站的背后感兴趣，都可以在本文中找到合适的方法。只要你愿意参与到网站的建设中，都可以加入到我们的社群中，成为 Mozilla Community 的一员。

更新文档

假如你在浏览 MDN 时，发现某篇文章有些问题，你完全可以试着自己动手去改正它。如果你对此感兴趣，这里有一篇参与指南供你参考。

MDN 的文件内容现在都存放在 GitHub 上的仓库中（翻译后的内容则存放在Translated Content中)，由一群志愿者和 Mozilla 员工共同管理。所以请放心大胆地参与到编辑中，会有人负责审核这些内容并帮助你修改它。

在 MDN 站内，也有许多文章可以为你提供帮助，比如：

  • 初识MDN中，简要介绍了参与方法。
  • 为 MDN 做贡献中，列出了许多等待参与的任务。

参与讨论

欢迎和我们交流！下面列出了一些途径，方便你和 MDN 的其他参与者们更好地交流。

在线实时聊天

我们使用 Matrix 来进行实时交流，主要是关于 MDN 以及其中的内容。我们随时都欢迎你的加入！

MDN Web Docs

一些日常的讨论，像网站的使用问题、网站上的文字内容、参与网站志愿编辑工作等，都可以在这个聊天室进行。如果你对网站上的文章内容有疑问或建议，或者希望在网站上看到或亲手创作一篇某个主题的新文章，又或者只是想和网站的内容创作者们有更多的交流，可以试着在这个聊天室寻找答案

在线论坛

MDN 的论坛上讨论的，大多是一些需要反复研究的内容。如果有想讨论的内容，向我们的邮箱——mdn@mozilla-community.org——发送邮件即可。在论坛注册帐号后，你还可以选择通过邮件来提醒你帖子下的新内容。

问题反馈

文档问题

如果你在文档中发现了问题，却不知道怎么修改，你可以向我们提交 issue，你遇到的任何问题都可以在这里反馈。

当然，如果你有时间，想试着自己来完善这个文档，我们也非常欢迎。当然，能够反馈你的发现问题，对我们也很有帮助！

网站问题

假如发现了网站本身的问题,或者对网站的功能任何的建议，你都可以直接联系 MDN 的开发团队。
