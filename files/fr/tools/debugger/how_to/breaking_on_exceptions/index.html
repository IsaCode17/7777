Pour que le débogueur s'arrête sur une exception, on cochera les cases suivantes dans la liste des points d'arrêt :
  • Pause sur les exceptions
  • Pause sur les exceptions interceptées

Lorsqu'une exception est déclenchée, la ligne où elle se produit est mise en évidence dans le panneau des sources avec une ligne ondulée rouge sous le code problématique. Une bulle d'information décrit l'exception. À partir de Firefox 80, un triangle permet de révéler la pile d'appel correspondante.
