Introduction à la mise en page en CSS
Cet article récapitule quelques-unes des fonctionnalités de mise en page CSS déjà abordées dans les modules précédents — telles que les diverses valeurs de {{cssxref ("display")}} — et présente certains des concepts que nous couvrirons dans de ce module.
Le flux normal
Par défaut, à moins d'apporter des modifications, les éléments se placent d'eux‑même sur les pages web selon un flux normal. Cet article explique les fondamentaux du cours normal pour permettre de comprendre comment le modifier.
Flexbox
Les boîtes flexibles (flexbox) sont une méthode de disposition sur une dimension pour organiser le contenu en lignes ou en colonnes. Les éléments s'étendent pour occuper l'espace ou rétrécissent dans des espaces plus étroits. Cet article couvre les bases de cette méthode. Après avoir étudié ce guide, vous pourrez tester vos compétences en flexbox pour assurer votre compréhension avant d'aller plus loin.
Les grilles
Les grilles CSS sont un système de disposition sur deux dimensions pour le Web. Elles permettent d'organiser le contenu en lignes et colonnes. Les grilles possèdent de nombreuses fonctionnalités qui facilitent la construction de dispositions complexes. Dans cet article, vous verrez tout ce qu'il faut pour commencer à manipuler les grilles puis vous pourrez tester vos compétences sur les grilles CSS avant d'aller plus loin.
Le flottement
Initialement destiné aux images « flottantes » autour desquelles on organisait le texte, la propriété float est devenu un outil fréquemment utilisé pour créer des dispositions en colonnes sur les pages web. Avec l'apparition des boîtes flexibles et des grilles, cette méthode a retrouvé son usage original, comme on pourra le voir dans cet article.
Le positionnement
Le positionnement permet de retirer les éléments du flux normal du document et d'altérer leur comportement. On peut ainsi les superposer les uns sur les autres ou les garder affichés à la même place dans la fenêtre du navigateur. Cet article explique les différentes valeurs de la propriété position et comment les utiliser.
La disposition en colonnes
Le module de spécification pour la disposition multi-colonnes fournit une méthode pour organiser le contenu en colonnes, à la façon d'un journal. Dans cet article, vous verrez comment utiliser cette fonctionnalité.
Le design responsive
Avec la diversification des tailles d'écran pour les appareils utilisés pour naviguer sur le Web, le concept de « responsive web design » (RWD) est apparu. Il s'agit d'un ensemble de pratiques permettant aux pages web d'arborer différentes dispositions et apparences pour s'adapter au mieux aux différentes largeurs d'écran, résolutions, etc. Dans cet article, nous verrons les principales techniques associées à ce concept.
Un guide pour s'initier aux requêtes média
Les requêtes média CSS (media queries en anglais) permettent d'appliquer certaines parties du CSS uniquement pour les navigateurs ou environnements qui respectent une règle donnée. Ainsi, on pourra déterminer le comportement « si la zone d'affichage du navigateur est plus large que 480 pixels ». Les requêtes média sont primordiales pour réaliser des sites et applications web responsifs. Elles permettent de détecter la taille de la zone d'affichage (viewport) mais aussi de fournir certaines informations sur l'environnement : est-ce que l'utilisateur utilise un écran tactile plutôt qu'une souris ? Dans cette leçon, vous commencerez par la syntaxe des requêtes média avant de voir comment les utiliser dans un exemple pour rendre un document responsif.
Les méthodes de disposition historique
Des systèmes de grille sont fréquemment utilisés dans des dispositions CSS. Avant l'apparition des grilles CSS, ces systèmes étaient implémentés grâce à des flottements ou via d'autres méthodes. Il fallait imaginer la disposition pour un nombre fixe de colonnes (par exemple 4, 6, ou 12), puis adapter le contenu des colonnes à l'intérieur de ces colonnes imaginaires. Dans cet article nous verrons comment ces méthodes passées fonctionnent afin que vous puissiez comprendre leur fonctionnement si vous travaillez sur un projet historique qui les utilise.
Prendre en charge les anciens navigateurs

Dans ce module, nous vous recommanderons d'utiliser flexbox et les grilles CSS comme méthodes pour vos dispositions. Toutefois, il peut y avoir des personnes qui utilisent votre site ou application à partir d'anciens navigateurs ou de navigateurs qui ne prennent pas en charge ces méthodes. De nouvelles fonctionnalités étant continuellement ajoutées au Web, les différents navigateurs implémentent à un rythme différent ces différentes fonctionnalités. Dans ces articles, nous verrons comment utiliser des techniques modernes sans pour autant laisser derrière les utilisateurs de navigateurs historiques.
