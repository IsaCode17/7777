Ce module prend la suite du module précédent : « Premiers pas en CSS ». Maintenant que vous avez acquis une familiarité avec le langage et sa syntaxe, que vous avez déjà expérimenté avec, il est temps d'approfondir. Dans ce module, nous verrons la cascade et l'héritage, les différents types de sélecteurs disponibles, les unités, le dimensionnement, la mise en forme des arrière-plans et des bordures, comment déboguer et plus encore.

L'objectif est de vous fournir les outils pour écrire du CSS qualifié et pour comprendre les briques théoriques essentielles avant d'aborder certains sujets plus spécifiques comme la mise en forme du texte et les dispositions en CSS.

Vous souhaitez devenir un développeur web front-end ?

Nous avons regroupé ensemble différents cours qui contiennent l'ensemble des informations à connaître pour cet objectif.

Démarrer le parcours

Prérequis

Avant de démarrer ce module, vous devriez :

  1. Savoir utiliser un ordinateur et naviguer sur le Web (consultation de sites web)
  2. Disposer d'un environnement de travail de base comme vu dans Installer les logiciels de base, et comprendre comment créer et gérer des fichiers comme abordé dans l'article Manipuler des fichiers.
  3. Connaître les bases de HTML, indiquées dans le module Introduction à HTML.
  4. Connaître les bases de CSS, indiquées dans le module Premiers pas en CSS.

Note : Si vous travaillez sur un appareil (ordinateur, tablette ou autre) sur lequel vous ne pouvez pas créer vos propres fichiers, vous pouvez essayer la plupart des exemples sur des sites de code en ligne tels que JSBin ou Glitch.

Guides

Ce module contient les articles suivants qui couvrent les aspects essentiels de CSS. Tout au long de ce module, vous aurez différents exercices qui vous permettront d'évaluer votre compréhension.

Cascade et héritage
L'objectif de cet article est d'améliorer votre compréhension de concepts fondamentaux en CSS : la cascade, la spécificité et l'héritage. Ceux-ci contrôlent la façon dont le CSS s'applique au document HTML et comment les conflits associés sont résolus.
Sélecteurs CSS
Il existe une large palette de sélecteurs CSS disponibles qui permettent de cibler finement les éléments à mettre en forme. Dans cet article et les sous-articles associés, nous verrons dans le détail les différents types de sélecteur, leur fonctionnement. Les sous-articles associés sont :
  • Sélecteurs de type, de classe et d'identifiant
  • Sélecteurs d'attribut
  • Pseudo-classes et pseudo-éléments
  • Combinateurs
Le modèle de boîte
En CSS, tout élément possède une boîte autour de lui. Comprendre le rôle de ces boîtes est primordial lorsqu'on souhaite organiser des dispositions en CSS ou aligner des éléments les uns par rapport aux autres. Dans cette leçon, nous verrons le modèle de boîte CSS afin que vous puissiez vous atteler à des tâches plus complexes tout en comprenant ce fonctionnement et la terminologie associée.
Arrière-plans et bordures
Dans cet article, nous verrons les effets créatifs qu'on peut obtenir avec les arrière-plans et les bordures en CSS. Que ce soit pour l'ajout de dégradé, les images d'arrière-plan, les angles arrondis, les arrière-plans et les bordures répondent à de nombreuses questions en CSS.
Gérer différentes directions de texte
Depuis quelques années, CSS a évolué afin de mieux prendre en charge la directionnalité du contenu, que ce soit pour les langues écrites de droite à gauche ou celles écrites du haut vers le bas (comme le japonais par exemple). Ces différentes directionnalités sont appelées des modes d'écriture. En avançant dans l'étude et la manipulation de CSS, comprendre les modes d'écriture vous sera utile, c'est pourquoi nous les verrons dans cet article.
Dépassement du contenu
Ici nous verrons un autre concept majeur de CSS : le dépassement. Le dépassement se produit lorsqu'il y a trop de contenu pour que celui-ci puisse tenir à l'intérieur d'une boîte. Dans ce guide, nous verrons ce qu'il en est et comment gérer les différents cas.
Unités et valeurs en CSS
Chaque propriété CSS possède une valeur ou un ensemble de valeurs autorisés. Dans cet article, nous verrons les valeurs et les unités les plus fréquemment utilisées.
Dimensionnement des objets en CSS
Dans les différentes parties de ce module, vous aurez vu différentes méthodes pour dimensionner des objets sur une page web en utilisant CSS. Comprendre comment définir la taille des différents aspects d'un document est important et nous résumerons ici les différents facteurs qui influent sur la taille d'un élément en CSS. Nous définirons également quelques termes sur le sujet pour vous aider par la suite.
Images, médias et formulaires
Dans cet article, nous verrons comment certains éléments spécifiques sont gérés en CSS. Les images et autres médias ainsi que les éléments de formulaire se comportent différemment des autres boîtes lorsqu'il s'agit de les mettre en forme avec CSS. Comprendre ce qu'il est possible ou non de faire pourra vous épargner de la frustration et vous faire gagner du temps. Cet article mettra en lumière les notions principales à connaître à ce sujet.
Mise en forme des tableaux
La mise en forme d'un tableau HTML n'est peut-être pas la tâche la plus avenante au premier regard. Il n'en reste pas moins que ça peut être nécessaire. Dans cet article, nous verrons comment afficher correctement des tableaux HTML et différentes techniques spécifiques associées.
Débogage du CSS
Lorsqu'on écrit du CSS, il arrive de rencontrer un problème où le résultat obtenu n'est pas celui attendu. Dans cet article, nous verrons comment analyser et déboguer un problème en CSS. Nous verrons comment utiliser les outils de développement intégrés aux navigateurs afin de déterminer ce qui pose problème.
Organiser son code CSS
Au fur et à mesure que vos feuilles de style s'agrandiront et que vous travaillerez sur des projets de plus grande ampleur, vous pourrez voir que la maintenance d'un énorme fichier CSS peut s'avérer délicate. Dans cet article, nous verrons certaines des bonnes pratiques pour écrire du CSS maintenable.

Évaluations

Vous souhaitez tester vos compétences en CSS ? Les évaluations suivantes vous permettront de vérifier votre compréhension pour les sujets abordés dans les guides listés ci-avant.

Compréhension des fondamentaux de CSS
Cette évaluation teste votre compréhension de la syntaxe de base, des sélecteurs, de la spécificité, du modèle de boîte, etc.
Créer une lettre à en-tête stylisé
Pour faire une bonne impression, écrire une lettre avec un en-tête raffiné peut être un bon point de départ. Dans cette évaluation, vous devrez créer un modèle d'une telle lettre.
Une boîte remarquable
Ici, vous pourrez pratiquer l'utilisation des arrière-plans et la mise en forme des bordures afin de créer une boîte qui ne laisse pas insensible.
