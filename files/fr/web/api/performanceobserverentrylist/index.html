{{APIRef("Performance Timeline API")}}

L'interface PerformanceObserverEntryList est une liste d'{{domxref("PerformanceEntry", "événements de performance","",1)}} qui ont été explicitement observés via la méthode {{domxref("PerformanceObserver.observe", "observe()")}}.

Remarque : cette interface est exposée à {{domxref("Window")}} et {{domxref("Worker")}}.

Méthodes

{{domxref("PerformanceObserverEntryList.getEntries","PerformanceObserverEntryList.getEntries()")}}
Retourne une liste d'objets explicitement observés de {{domxref("PerformanceEntry")}} en fonction du filtre donné.
{{domxref("PerformanceObserverEntryList.getEntriesByType","PerformanceObserverEntryList.getEntriesByType()")}}
Retourne une liste d'objets explicitement observés {{domxref("PerformanceEntry")}} du type d'entrée donné.
{{domxref("PerformanceObserverEntryList.getEntriesByName","PerformanceObserverEntryList.getEntriesByName()")}}
Renvoie une liste d'objets explicitement observés {{domxref("PerformanceEntry")}} en fonction du nom et du type d'entrée donnés.

Exemple

// Crée un observateur pour tous les types d'événements de performance
// la liste est de type PerformanceObserveEntryList
const observe_all = new PerformanceObserver(function(list, obs) {
  let perfEntries = list.getEntries();
  for (let i = 0; i < perfEntries.length; i++) {
    print_perf_entry(perfEntries[i]);
    // faire quelque chose avec
  }
})
