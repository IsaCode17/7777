{{ApiRef("HTML DOM")}}

L'attribut dir obtient ou définit la direction d'écriture du texte pour le contenu de l'élément courant.

La directionnalité d'écriture d'un élément est la direction dans laquelle le texte va (pour la prise en charge de différents systèmes linguistiques). Les langues arabes et l'hébreu sont des langues typiques utilisant la directionnalité RTL.

Une image peut avoir sa propriété dir définie sur « rtl », auquel cas les attributs HTML title et alt seront formatés et définis comme « rtl ».

Lorsqu'un tableau a son dir défini sur « rtl », l'ordre des colonnes est disposé de droite à gauche.

Lorsqu'un élément a sa direction définie sur « auto », la direction de l'élément est déterminée sur la base de son premier caractère de direction forte, ou par défaut sur la direction de son élément parent.

Note :

Les navigateurs peuvent permettre aux utilisateurs de modifier la directionnalité des <input> et <textarea> afin de faciliter la création de contenu. Chrome et Safari proposent une option de directionnalité dans le menu contextuel des champs de saisie, tandis qu'Internet Explorer et Edge utilisent les combinaisons de touches Ctrl + Shift gauche et Ctrl + Shift droit. Firefox utilise Ctrl/Cmd + Shift + X mais ne met PAS à jour la valeur de l'attribut dir.

Syntaxe

let currentWritingDirection = elementNodeReference.dir;
elementNodeReference.dir = newWritingDirection;
  • currentWritingDirection est une variable de type chaîne de caractères représentant le sens d'écriture du texte de l'élément courant.
  • newWritingDirection est une variable de type chaîne de caractères représentant la valeur du sens d'écriture du texte.

Les valeurs possibles pour dir sont ltr, pour de gauche à droite (left-to-right), et rtl, pour de droite à gauche (right-to-left).

Exemple

let para = document.getElementById("para1");
para.dir = "rtl";
// change la direction du texte sur un paragraphe identifié par "para1"
