{{APIRef("HTML DOM")}}

La méthode HTMLElement.click() simule un clic de souris sur un élément.

Lorsque click() est utilisé avec des éléments pris en charge (comme un <input>), il déclenche l'événement de clic de l'élément. Cet événement remonte ensuite jusqu'aux éléments situés plus haut dans l'arbre du document (ou chaîne d'événements) et déclenche leurs événements de clic.

Syntaxe

element.click()

Exemple

Simulez un clic de souris lorsque vous déplacez le pointeur de la souris sur une case à cocher :

HTML

<form>
  <input type="checkbox" id="myCheck" onmouseover="myFunction()" onclick="alert('l'événement de clic s'est produit')">
</form>

JavaScript

// Au passage de la souris, exécuter myFunction
function myFunction() {
  document.getElementById("myCheck").click();
}
