{{APIRef("Performance Timeline API")}}

La méthode takeRecords() de l'interface {{domxref('PerformanceObserver')}} renvoie la liste actuelle des {{domxref("PerformanceEntry", "entrées de performances","",1)}} stockés dans l'observateur de performance, en la vidant.

Syntaxe

let performanceEntryList = performanceObserver.takeRecords();

Paramètres

Aucun.

Valeur de retour

Une liste d'objets {{domxref("PerformanceEntry")}}.

Exemple

const observer = new PerformanceObserver(function(list, obj) {
  let entries = list.getEntries();
  for (let i=0; i < entries.length; i++) {
    // Traiter les événements "mark" et "frame"
  }
});
observer.observe({entryTypes: ["mark", "frame"]});
let records = observer.takeRecords();
console.log(records[0].name);
console.log(records[0].startTime);
console.log(records[0].duration);
