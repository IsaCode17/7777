{{APIRef("User Timing API")}}

PerformanceMark est une interface abstraite pour les objets {{domxref("PerformanceEntry")}} avec un {{domxref("PerformanceEntry.entryType", "entryType")}} « mark ».Les entrées de ce type sont créées en appelant {{domxref("Performance.mark", "performance.mark()")}} pour ajouter un {{domxref("DOMHighResTimeStamp")}} nommé (le marqueur) à la chronologie des performances du navigateur.

{{InheritanceDiagram}}

{{AvailableInWorkers}}

Propriétés

Cette interface n'a pas de propriétés mais elle étend les propriétés de {{domxref("PerformanceEntry")}} suivantes en qualifiant/contraignant les propriétés comme suit :

{{domxref("PerformanceEntry.entryType")}}
Retourne « mark ».
{{domxref("PerformanceEntry.name")}}
Retourne le nom donné au marqueur lorsqu'il a été créé via un appel de {{domxref("Performance.mark()", "performance.mark()")}}.
{{domxref("PerformanceEntry.startTime")}}
Retourne le {{domxref("DOMHighResTimeStamp")}} à partir du moment où {{domxref("Performance.mark()", "performance.mark()")}} a été appelé.
{{domxref("PerformanceEntry.duration")}}
Retourne « 0 ». (Un marqueur n'a pas de durée.)

Méthodes

Cette interface n'a pas de méthodes.
