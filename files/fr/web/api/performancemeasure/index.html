{{APIRef("User Timing API")}}

PerformanceMeasure est une interface abstraite pour les objets {{domxref("PerformanceEntry")}} ayant un {{domxref("PerformanceEntry.entryType", "entryType")}} « measure ». Les entrées de ce type sont créées en appelant {{domxref("Performance.measure", "performance.measure()")}} pour ajouter un {{domxref("DOMHighResTimeStamp")}} nommé (la mesure) entre deux marqueurs à la chronologie des performances du navigateur.

{{InheritanceDiagram}}

{{AvailableInWorkers}}

Propriétés

Cette interface n'a pas de propriétés mais elle étend les propriétés de {{domxref("PerformanceEntry")}} suivantes en qualifiant/contraignant les propriétés comme suit :

{{domxref("PerformanceEntry.entryType")}}
Retourne « measure ».
{{domxref("PerformanceEntry.name")}}
Retourne le nom donné à la mesure lorsqu'elle a été créée via un appel de {{domxref("Performance.measure()", "performance.measure()")}}.
{{domxref("PerformanceEntry.startTime")}}
Retourne un {{domxref("DOMHighResTimeStamp", "timestamp")}} donné à la mesure lorsqu'elle a été appelée par {{domxref("Performance.measure()", "performance.measure()")}}.
{{domxref("PerformanceEntry.duration")}}
Retourne un {{domxref("DOMHighResTimeStamp")}} correspondant à la durée de la mesure (généralement, l'horodatage de fin de la mesure moins l'horodatage de début).

Méthodes

Cette interface n'a pas de méthodes.
