La propriété csp de l'interface {{domxref("HTMLIFrameElement")}} spécifie la Content Security Policy (ou stratégie de sécurité du contenu) qu'un document embarqué doit accepter pour pouvoir s'auto-appliquer.

Syntaxe

var csp = HTMLIFrameElement.csp
HTMLIFrameElement.csp = csp

Valeur

Une content security policy (stratégie de sécurité du contenu).
