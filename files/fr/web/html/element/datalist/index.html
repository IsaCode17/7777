{{HTMLRef}}

L'élément HTML <datalist> contient un ensemble d'éléments <option> qui représentent les valeurs possibles pour d'autres contrôles.

{{EmbedInteractiveExample("pages/tabbed/datalist.html", "tabbed-standard")}}
Catégories de contenu Contenu de flux, contenu phrasé.
Contenu autorisé Soit du contenu phrasé ou zéro ou plus d'éléments <option>.
Omission de balises Aucune, la balise d'ouverture et la balise de fermeture sont obligatoires.
Parents autorisés N'importe quel élément qui accepte du contenu phrasé.
Rôle ARIA implicite listbox
Rôles ARIA autorisés Aucun.
Interface DOM HTMLDataListElement

Attributs

Cet élément ne possède que les attributs universels, communs à tous les éléments.

Exemples

Liste de données de base

<label for="monNavigateur">Veuillez choisir un navigateur parmi ceux-ci :</label>
<input list="navigateurs" id="monNavigateur" name="monNavigateur"/>
<datalist id="navigateurs">
  <option value="Chrome">
  <option value="Firefox">
  <option value="Internet Explorer">
  <option value="Opera">
  <option value="Safari">
  <option value="Microsoft Edge">
</datalist>

Personnaliser les styles des listes de données

Le style de la liste produite par un élément <datalist> dépend de la plate-forme, donc si vous voulez personnaliser l'affichage des options, vous devez mettre en œuvre votre propre solution personnalisée qui remplace le comportement par défaut. L'exemple ci-dessous fournit une solution simple pour cela. Notez comment nous n'avons pas spécifié le <datalist> id à l'intérieur de l'attribut <input> list dans ce cas, car cela empêche l'affichage spécifique au navigateur des éléments de la liste de données de se mettre en place. Au lieu de cela, nous définissons la valeur <datalist> sélectionnée dans le <input> via JavaScript.

Note :

Puisque ::after n'est pas autorisé sur les éléments <input>, si vous voulez reproduire l'icône de la flèche vers le bas, vous devrez envelopper l'élément <input> dans un autre élément sur lequel vous pourrez accrocher le style (ou une autre solution appropriée).

HTML

<input
  list=""
  name="option"
  id="input"
  autocomplete="off"
>

<datalist id="datalist">
  <option>Carottes</option>
  <option>Pois</option>
  <option>Haricots</option>
</datalist>

CSS

datalist {
  position: absolute;
  background-color: lightgrey;
  font-family: sans-serif;
  font-size: 0.8rem;
}

option {
  background-color: #bbb;
  padding: 4px;
  margin-bottom: 1px;
  cursor: pointer;
}

JavaScript

input.onfocus = function () {
  datalist.style.display = 'block';
}

for (let option of datalist.options) {
  option.onclick = function () {
    input.value = this.value;
    datalist.style.display = 'none';
  }
}

datalist.style.width = input.offsetWidth + 'px';
datalist.style.left = input.offsetLeft + 'px';
datalist.style.top = input.offsetTop + input.offsetHeight + 'px';
