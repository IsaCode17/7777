--- title: Accentuation slug: Glossary/accent translation_of: 'Glossary/accent' ---

Une accentuation est une couleur généralement vive qui contraste avec les couleurs d'arrière-plan et de premier plan plus utilitaires d'un jeu de couleurs. Celle-ci sont présentes dans le style visuel de nombreuses plates-formes (mais pas toutes).

Sur le web, une accentuation est parfois utilisé dans les éléments {{HTMLElement("input")}} pour la partie active du contrôle, par exemple l'arrière-plan d'une case cochée (« checkbox » en anglais).

CSS lié à l'accentuation

Vous pouvez définir la couleur d'accentuation pour un élément donné en définissant la propriété CSS {{cssxref("accent-color")}} de l'élément sur la valeur {{cssxref("<color>")}} appropriée.

Éléments HTML pouvant avoir une accentuation

  • <input type="checkbox">
  • <input type="radio">
  • <input type="range">
