Dans un navigateur, le chrome correspond à n'importe quel aspect visible d'un navigateur en dehors des pages web elles-mêmes (par exemple la barre d'outils, la barre de menu, les onglets). Il ne doit pas être confondu avec le navigateur Google Chrome.
