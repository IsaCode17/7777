Web terbuka dibangun menggunakan sejumlah teknologi. Di bawah ini untuk masing-masingnya, Anda akan menemukan tautan ke materi referensi kami.

Teknologi Web Inti

Referensi antarmuka API Web
Materi referensi untuk setiap antarmuka yang terdiri dari API Web, termasuk DOM dan semua API dan antarmuka terkait yang dapat Anda gunakan untuk membuat konten dan aplikasi Web. Daftar ini mencakup semua antarmuka, disusun menurut abjad.
Web APIs
Daftar API individu dan rangkaian teknologi yang membentuk API Web secara keseluruhan.
HTML
HyperText Markup Language adalah bahasa yang digunakan untuk mendeskripsikan dan menentukan konten halaman Web.
CSS
Cascading Style Sheets digunakan untuk mendeskripsikan tampilan konten Web.
SVG
Scalable Vector Graphics memungkinkan Anda mendeskripsikan gambar sebagai kumpulan vektor dan bentuk agar dapat diskalakan dengan mulus terlepas dari ukurannya.
MathML
Bahasa Markup Matematika memungkinkan untuk menampilkan persamaan dan sintaks matematika yang kompleks.
