Ringkasan

Mengembalikan nama file. Untuk alasan keamanan, jalur dikecualikan dari properti ini.

Properti ini tidak digunakan lagi. Gunakan {{domxref("File.name")}} sebagai gantinya.

Syntax

var name = instanceOfFile.fileName

Value

string
