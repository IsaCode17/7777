Situs Web Dinamis – Pemrograman sisi server topik adalah serangkaian modul yang menunjukkan cara membuat situs web dinamis; situs web yang memberikan informasi khusus sebagai tanggapan atas permintaan HTTP. Modul menyediakan pengenalan umum untuk pemrograman sisi server, bersama dengan panduan tingkat pemula khusus tentang bagaimana menggunakan kerangka web Django (Python) dan Express (Node.js / JavaScript) untuk membuat aplikasi dasar.

Sebagian besar situs web utama menggunakan beberapa jenis teknologi sisi server untuk secara dinamis menampilkan data yang berbeda sesuai kebutuhan. Misalnya, bayangkan berapa banyak produk yang tersedia di Amazon, dan bayangkan berapa banyak postingan yang telah ditulis di Facebook? Menampilkan semua ini menggunakan halaman statis yang sama sekali berbeda akan sangat tidak efisien, jadi situs seperti itu menampilkan template statis (dibuat menggunakan HTML, CSS, and JavaScript), dan kemudian secara dinamis memperbarui data yang ditampilkan di dalam template tersebut saat diperlukan, misal saat Anda ingin melihat produk lain di Amazon.

Dalam dunia modern pengembangan web, mempelajari tentang pengembangan sisi server sangat disarankan.

Jalur pembelajaran

Memulai dengan pemrograman sisi server biasanya lebih mudah daripada dengan pengembangan sisi klien, karena situs web dinamis cenderung melakukan banyak operasi yang sangat mirip (mengambil data dari database dan menampilkannya di halaman, memvalidasi data yang dimasukkan pengguna dan menyimpannya dalam database, memeriksa izin pengguna dan memasukkan pengguna, dll.), dan dibuat menggunakan kerangka kerja web yang membuat ini dan operasi server web umum lainnya mudah.

Pengetahuan dasar tentang konsep pemrograman (atau bahasa pemrograman tertentu) berguna, tetapi tidak penting. Demikian pula, keahlian dalam pengkodean sisi klien tidak diperlukan, tetapi pengetahuan dasar akan membantu Anda bekerja lebih baik dengan pengembang yang membuat "front end" web sisi klien Anda.

Server-side website programming first steps
Modul ini memberikan informasi agnostik teknologi server tentang pemrograman situs web sisi server, termasuk jawaban atas pertanyaan mendasar tentang pemrograman sisi server - "apa itu", "perbedaannya dengan pemrograman sisi klien", dan "mengapa begitu berguna "- dan ikhtisar dari beberapa kerangka kerja web sisi server yang lebih populer dan panduan tentang cara memilih yang paling sesuai untuk situs Anda. Terakhir kami memberikan bagian pengantar tentang keamanan server web.
Django Web Framework (Python)
Django adalah kerangka kerja sisi server yang sangat populer dan berfitur lengkap, ditulis dengan Python. Modul menjelaskan mengapa Django adalah kerangka server web yang bagus, bagaimana menyiapkan lingkungan pengembangan dan bagaimana melakukan tugas-tugas umum dengannya.
Express Web Framework (Node.js/JavaScript)
Express adalah kerangka kerja web populer, ditulis dalam JavaScript dan dihosting dalam lingkungan runtime node.js. Modul ini menjelaskan beberapa manfaat utama kerangka kerja ini, cara menyiapkan lingkungan pengembangan Anda, dan cara melakukan tugas pengembangan dan penerapan web umum.
