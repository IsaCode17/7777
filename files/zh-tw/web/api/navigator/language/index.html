{{APIRef("HTML DOM")}}

Navigator.language 為只讀屬性，返回一個表示用戶偏好語言（通常是瀏覽器界面語言）的字符串。

語法

const lang = navigator.language

值

一个 {{domxref("DOMString")}}. lang存儲了一個BCP 47中定義的語言版本。 一些有效的語言版本代碼示例包括：「en」、「en-US」、「fr」、「fr-FR」、「es-ES」等。

請注意，在iOS 10.2之前的Safari中，返回的語言版本代碼是小寫的：「en-us」、「fr-fr」等。

示例

if (/^en\b/.test(navigator.language)) {
  doLangSelect(window.navigator.language);
}
