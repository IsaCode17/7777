{{JSRef}}

trim() 方法會移除字串起始或結尾處的空白格。 本文中的空白格泛指所有空格字元及換行字元。
空格字元如：空格字元 ( Space )，欄標字元 ( Tab )、不換行空格 ( No-break space ) …… 等等。
換行字元如：LF ( Line Feed )、CR ( Carriage Return ) …… 等等。

語法

trim()

回傳值

回傳一個新的字串，其為把 str 起始及結尾處的空白格移除後的值。

如果 str 的起始及結尾處沒有任何的空白格，此方法不會拋出任何異常，且仍然會回傳一個新的字串，但它的值與 str 相同。

如果只是想要去除字串起始處或結尾處其中之一的空白格，那麼可以選擇使用 {{jsxref("String.prototype.trimStart()", "trimStart()")}} 或者 {{jsxref("String.prototype.trimEnd()", "trimEnd()")}}。

Polyfill

在任何其他的程式碼被執行之前，先執行以下的程式碼，它將會在瀏覽器本身未支援 trim() 的方法時創造它。 

if (!String.prototype.trim) {
  String.prototype.trim = function () {
    return this.replace(/^[\s\uFEFF\xA0]+|[\s\uFEFF\xA0]+$/g, '');
  };
}

範例

如何使用 trim()

以下例子會將原先字串前後的空白格移除，最後印出 'foo'：

var orig = '   foo  ';
console.log(orig.trim()); // 'foo'
