# macro 안내서

## 명세서

명세서 영역은 [Replace old-style compatibility/specification tables with new macro](https://github.com/mdn/translated-content/issues/5618)에 따라 아래와 같이 작성합니다.

```
## 명세서

{{Specifications}}
```
